# 🌍 我的小世界（My Little World）

一个用 **three.js** 做的、给 5 岁小朋友玩的**低多边形恐龙世界模拟器**。
在一座软软的小岛上，孩子可以**造山、挖海、种树花草、放置恐龙、切换白天黑夜和天气**，
配有**全部用代码实时合成的背景音乐和音效**（没有任何外部音频文件）。

进入游戏时可以选择四种世界：

- **恐龙乐园**：带树林、花田、食草龙和翼龙的默认场景。
- **高山峡谷**：适合观察追逐和建造山谷家园。
- **群岛世界**：由多座岛屿组成，可以继续挖海或连接陆地。
- **从零开始**：宽阔的空岛，适合完全自由创作。

## ✨ 玩法

底部一排大圆角按钮就是孩子的“神力”：

| 按钮 | 作用 |
| --- | --- |
| ⛰️ 造山 | 在小岛上按住拖动，地面会鼓成圆圆的山，越按越高 |
| 🌊 挖海 | 按住拖动把地面压下去，低处自动变成海/湖 |
| 🌳 种树 | 点一下长出一棵树（拖动可以种一排小树林） |
| 🌷 种花 | 点一下开出一朵彩色小花 |
| 🦕 三角龙 / 腕龙 / 剑龙 | 放置食草恐龙；饿了会寻找并吃掉树或花 |
| 🦖 霸王龙 / 迅猛龙 | 放置食肉恐龙；只捕食体型更小的地面恐龙 |
| 🥚 窃蛋龙 | 寻找并吃掉恐龙蛋；其他恐龙不会吃蛋 |
| 🪽 翼龙 | 放一只在天空巡航的翼龙，不参与进食和捕食 |

恐龙会在约一分钟内自然长大，吃到食物后还会继续成长，并且没有体型上限。每次进食后会拉下一小堆粑粑，再休息一段时间，不会连续进食；找不到食物也不会饿死。成熟恐龙会自动搭建同物种巢穴，前往巢穴产下对应的蛋；翼龙会从空中把蛋投进巢里。

恐龙数量没有硬上限。右上角显示 `当前恐龙数 / 100`，80 只后变黄、100 只后变红，仅用于提示设备可能出现性能压力，不会删除或阻止新增恐龙。

右上角按钮：

- 🌗 切换 **白天 → 黄昏 → 夜晚**（天空、太阳/月亮、星星都会变）
- 🌧️ 下雨开/关
- 🌈 出现彩虹
- 🔊 / 🔇 声音开关
- 🔄 重新开始一个新世界

**操作说明**：一根手指（或鼠标左键）在小岛上用工具；点在**天空**上拖动可以**转动视角**，
两根手指可以缩放并旋转。

## 🚀 运行

需要 Node.js（建议 18+）。

```bash
npm install      # 安装依赖（three + vite）
npm run dev      # 本地开发，自动打开浏览器
npm run build    # 打包到 dist/，可部署到任意静态服务器
npm run preview  # 预览打包结果
```

## 🧱 项目结构

```
src/
├── main.js              # 启动 + 主循环，装配所有系统
├── style.css            # UI 样式（粉彩、大圆角按钮、开始页）
├── constants.js         # 全局常量（世界尺寸、海平面、数量上限）
├── utils.js             # 缓动 / 随机 / 噪声等小工具
├── world/
│   ├── Scene.js         # 渲染器 / 相机 / 控制器 / 灯光 / 雾
│   ├── Terrain.js       # 高度网格、场景预设、造山挖海笔刷、按高度上色
│   ├── Water.js         # 半透明海面 + 波光
│   └── Sky.js           # 渐变天空、太阳/月亮、星星、云、昼夜过渡
├── entities/
│   ├── Tree.js          # 程序化的树与花（带“长出来”动画）
│   ├── Dinosaur.js      # 七种恐龙模型 + 漫游、飞行、进食、产蛋与成长
│   └── Ecosystem.js     # 物种巢穴、恐龙蛋、孵化流程和粪便实体
├── systems/
│   ├── Tools.js         # 工具定义与当前选择
│   ├── Input.js         # 指针/触摸 → 射线 → 执行当前工具
│   ├── Weather.js       # 下雨粒子 + 彩虹
│   └── Audio.js         # Web Audio 合成：背景旋律 + 各种音效
└── ui/
    └── Toolbar.js       # 底部工具栏 + 顶部动作/声音/重置按钮
```

## 🎨 设计取向

- **低多边形萌系**：圆润山丘、马卡龙粉彩、flat shading、胖胖可爱的树和恐龙。
- **5 岁友好**：极少文字、超大图标、点一下就有满足感、**没有失败/分数**，纯创造与探索。
- **零素材依赖**：声音全部由 Web Audio 合成，可离线、无版权问题。
